# e-plantShopping

## Project Overview

e-plantShopping is a React and Redux-based online plant shopping application developed as a final project. The application allows users to browse a variety of house plants, add products to a shopping cart, update quantities, remove items, and view the total cart value.

## Features

* Landing page with navigation to the product catalog
* Product listing page displaying different categories of plants
* Add-to-cart functionality
* Shopping cart with quantity management
* Dynamic cart item count in the navigation bar
* Redux state management for cart operations
* Responsive and user-friendly interface

## Technologies Used

* React
* Redux Toolkit
* JavaScript
* CSS
* Vite
